# Máquinas Secuenciales

Podemos resumir los esquemas de Recorrido utilizando el método *CTA*.

## Primer Modelo

**C**omenzar  
**T**ratamiento Inicial  
**C**omprobar (¿Es marca Fin?)  
**T**ratamiento Elemento Actual  
**A**vanzar

## Segundo Modelo

**I**niciar  
**T**ratamiento Inicial  
**A**vanzar  
**T**ratamiento Elemento Actual  
**C**omprobar (¿Es Último?)
